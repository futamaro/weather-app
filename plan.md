# 天氣預報與生活助手 (Weather Forecast & Life Assistant)

這是一個基於 Python 開發的桌面天氣預報與生活助手應用程式。本專案專為學期末作業設計，完全使用 Python 內建的標準庫開發，不需要透過 `pip` 安裝任何第三方套件即可直接運行。

---

## ✨ 核心特色與功能
1. **純標準庫實現 (Zero Dependencies)**
   * GUI 介面：使用 `tkinter` 與 `tkinter.ttk` 實作。
   * 網路連線：使用 `urllib.request` 連接網路 API。
   * 圖片處理：利用系統內建的 Unicode 天氣符號（如 ☀️、🌧️、☁️）代替外部 PNG 圖示，免除安裝 `Pillow` 的限制。
2. **現代扁平化設計 (Sleek Flat UI - Light Mode)**
   * 採用明亮簡約的配色（天空藍與乾淨白），捨棄了傳統 Tkinter 立體復古邊框，使用扁平卡片化視覺排版。
3. **雙模式數據引擎 (Dual Mode Engine)**
   * **模擬數據模式**（預設）：若無 API 金鑰，系統將以城市名稱與日期哈希作為種子，產生流暢、合理的模擬天氣與 3 天預報，非常適合在無網路或無金鑰的場合展示。
   * **真實氣象模式**：在「設定」中填入 OpenWeatherMap API 金鑰後，將自動連網下載最真實的即時天氣與未來 3 天預報。
4. **生活助手建議系統**
   * 根據當前溫度、濕度、風速與天氣狀況，提供：
     * **👕 穿衣指南**（短袖、薄外套、大衣、羽絨服建議）
     * **🌂 雨具防護**（是否攜帶雨傘或雨衣建議）
     * **🏃 戶外運動**（是否適合慢跑、騎單車、室內運動建議）
     * **🧴 健康生活**（防曬、保濕、室內除濕建議）
5. **非同步連網與多執行緒 (Async Threading)**
   * 連網查詢與載入時使用背景執行緒處理，介面絕不卡死，維持流暢體驗。
6. **城市管理系統**
   * 支援搜尋任意城市（真實模式下支援全球英文城市搜尋）。
   * 支援將常查詢的城市「加入常用清單」，快速點擊即可切換。
   * 自動儲存設定檔 `weather_config.json`，在下次開啟時自動讀取最愛城市與設定。

## 🚀 快速開始

### 1. 執行環境
* 作業系統：Windows 10/11, macOS, 或 Linux
* Python 版本：Python 3.6+ (內建 Tkinter)

### 2. 執行指令
在終端機 (Terminal / Command Prompt) 中進入此專案目錄，輸入以下指令即可執行：
```bash
python weather_app.py
```

---

## ⚙️ 設定真實氣象 API (選用)
1. 前往 [OpenWeatherMap 官網](https://openweathermap.org/) 免費註冊一個帳號。
2. 進入 API Keys 頁面，複製您的 **API Key**。
3. 開啟本 App，點擊左下角的 **⚙️ 設定與 API 金鑰**。
4. 貼上您的 API 金鑰，並點擊 **儲存設定**。
5. 成功後，右上角狀態會從「模擬模式」切換成「真實氣象模式」。

*註：OpenWeatherMap 免費 API 金鑰註冊後通常需要等待 10~30 分鐘才會正式啟用。*

## 📁 檔案結構說明
* `weather_app.py`：主程式檔，包含 UI 繪製、非同步連線、Mock 天氣引擎與生活建議規則。
* `weather_config.json`：自動產生的本機設定檔，用於儲存最愛城市清單、溫度單位偏好及 API 金鑰。
* `README.md`：本說明文件。
